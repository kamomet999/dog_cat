# 課金モデル — フリーミアム（無料＋¥500で全品種）

- 作成 2026-06-12 ／ 発案者決定。**このファイルが課金まわりの「正」。**
- 旧方針「完全無料・全種むりょう」からの更新。理由＝持続可能な個人開発のための収益源。

## 2つの「課金」を区別する（最重要）

| | 内容 | 可否 |
|---|---|---|
| ❌ 延命・復活の課金 | 死んだ子を金で生き返らせる／寿命を金で延ばす | **永久になし**（DESIGN.md 原則2「責めない」を壊す） |
| ✅ コレクション拡張の課金 | メジャー種は無料／¥500で全公式品種を解放 | **採用**（延命ではないので世界観と両立） |

→ 「広告ゼロ」「延命・復活の課金なし」という約束は**不変**。変わったのは「全種が無料」だった点だけ。

## ティア

| | 無料 | プレミアム（¥500 買い切り） |
|---|---|---|
| 収録品種 | **メジャーな犬30種＋猫30種＝計60種**（`breeds.js` の tier 無し） | **全公式品種**（犬は最終 200〜300、猫は公認種＋毛色変種） |
| 抽選 | 無料プールのみ（`Breeds.roll(rnd, luck, false)`） | 全品種（`...true`） |
| 図鑑の目標 | 30種コンプ | 全種コンプ |
| 広告 | なし | なし |
| 追加課金 | — | なし（買い切り1回のみ） |

- 商品ID: `inuneko_premium_unlock` ／ 表示 `¥500`（`Breeds.PREMIUM`）
- 実装状態: **仕組みは完成**（フラグ `state.premium`・`Engine.unlockPremium()`・図鑑のCTA/解放表示・購入モーダル）。
  プレミアム品種の**本データ拡充（可愛い人気種）は後続**。現状は先行サンプル12種（犬6・猫6）。
- **猫の現実**: 公認の猫種は実数で ~70〜100 程度。「猫200〜300」は毛色・柄変種（キジトラ/三毛/サバトラ 等の方式）を含めて数を確保する。犬は公認種だけで200〜300届く。

## 課金処理（実購入）

- **MVP/web**: 購入ボタンで即 `Engine.unlockPremium()`（ストア接続なし。検証用）。
- **実機（後続）**: Capacitor の課金プラグイン（RevenueCat もしくは StoreKit/Play Billing）で
  `inuneko_premium_unlock`（非消費型）を購入 → 成功コールバックで `unlockPremium()`。
  「購入をふくげん（Restore）」も同プラグインの restore に接続する。
- ストア側の商品登録は Apple/Google の有料アカウント確定後のタスク。
