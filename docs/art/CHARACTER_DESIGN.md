## 2. 設計値 — 基準との対応表

すべての数値は原画座標系（viewBox 200×200）。各行に根拠の基準番号を付す。

### 2.1 シルエット（DEEP_DIVE §1）

| 項目 | 設計値 | 基準 |
|---|---|---|
| 体型 | 頭と体が一体の「もちブロブ」。曲率は**頭頂→こめかみ(締)→頬(張)→首のくびれ(締)→胴(張)→接地(潰)**とリズムを持たせる。等速曲線を禁止 | §1-3 曲率リズム |
| 頭:胴 | **頭が最大幅（頬112）・胴はその約0.9倍（104）**。胴が頭より太くなると「だるま」化して幼形が崩れる（2026-06-11 発案者指摘で確定） | §1-4, ベビースキーマ |
| 前足 | 体に埋めず**接地線から手前にはみ出す**（cy178・ry10 > 底辺182）。指スジ2本で「おてて」を明示 | §7 届かなさ |
| 凸率 | シルエット÷凸包 ≧ **0.92**（凹みは耳の付け根と腰のくびれ計2〜3箇所のみ） | §1-2 |
| 重心 | 下から **0.35〜0.45**（質量の6割が下半分） | §1-4 |
| 接地 | 底辺幅 ≧ 最大幅×**0.8**・底はほぼ平らに潰す（自重で広がるおもち） | §1-4 |
| 突起 | 基本形は**耳2＋尾1＋腕2**まで（腕は擬人化の必須記号として例外許容。小さく・短く「届かない」こと）。脚はシルエット内＋接地線の足先で表す | §1-5, §7 |
| 非対称 | 右側の輪郭を**1〜2単位（約1%）**だけずらす。尾は片側 | §11-4「1%の生」 |

### 2.2 顔面構成（DEEP_DIVE §2／RUBRIC 指標2）

| 項目 | 設計値 | 基準 |
|---|---|---|
| Tゾーン凝縮 | 目と口を含む最小円 ÷ 顔幅 = **55〜65%**（実装値: クラスタ径70 ÷ 頬幅112 ≒ 62%） | §2-1 |
| 目の高さ | y=98（顔の中心より下・頭頂42〜接地184の上から約38%） | §2-1, ベビースキーマ |
| おでこ | y42〜86 は**無地**（模様も置かない。タン眉のみ目の直上1目分離して可） | §2-2 |
| 三角形 | 両目中心(77/123, y98)と口(100, y114)の三角形: 高さ16÷底辺46 ≒ **0.35→縦圧縮ぎみの幼形** | §2-3 |
| 頬 | 輪郭の最大張り出しを**目より下（y≒110）**に置き、ほっぺ彩色はその位置に重ねる | §2-4 |

### 2.3 目（DEEP_DIVE §3／RUBRIC 指標3）

| 項目 | 設計値 | 基準 |
|---|---|---|
| 大きさ | r=11.5（目径23 ÷ 顔幅112 ≒ **20%**） | RUBRIC 指標3 |
| 虹彩 | 楕円 rx=0.85r・**ry=0.80r**（可視高さ比0.94＝下をわずかに切った微笑みの目） | §3-2 |
| 瞳孔 | 0.58r（瞳孔÷虹彩 ≒ **68%**） | §3-1（55〜70%） |
| 視線 | 瞳孔とハイライトを**内側へ1.2**（顔幅の約1%）寄せる | §3-3 |
| ハイライト | 主1（径=虹彩の**26%**・左上＝光源側で全キャラ統一）＋副1（対角・小・80%透過）。**3個以上禁止** | §3-1, §3-4 |

### 2.4 線（DEEP_DIVE §4／RUBRIC 指標5）

| 項目 | 設計値 | 基準 |
|---|---|---|
| 階層 | 外輪郭 **3.2** ＞ 顔パーツ 2.2〜2.4（=0.7倍） ＞ 模様 ≦1.6（タビー縞は2.2の塗り扱い） | §4-1 |
| 色 | 輪郭は茶系 rgba(74,52,30,.5)。**#000禁止**。最暗 #241a12 | §4-2 |
| 質 | 均一太さ＋丸端。入り抜きはしない（量産30種の文法） | §4-4 |

### 2.5 色（DEEP_DIVE §5／RUBRIC 指標4）

地色2（color/color2）＋記号色2（目・ほっぺ#ffb3a0）まで。柄種のみ+1。顔まわりが最明（暗色種はクリーム補正）。影は描かない。

### 2.6 記号と感情（DEEP_DIVE §9）

ω口（猫）／へにゃ口（犬）常時。ほっぺ=体温（悲しみ時のみ消す）。感情変化は**目＋口の2チャンネル同時**。汗・怒りマーク禁止。

### 2.7 品種らしさ＝1点豪華主義（DEEP_DIVE §10／RUBRIC 指標6）

| 優先 | チャンネル | 実装 |
|---|---|---|
| 1 | 耳 | prick / bigprick / flop / round / fold |
| 2 | 柄1記号 | ハチワレV・シャムポイント・水玉 等8種 |
| 3 | 色 | 黒猫・白猫 等 |
| 4 | 尾 | **柴犬=巻き尾（curl）を新設**（最強の識別子） |

検収: 名前を隠して30種を3秒ずつ → **7割が種名を言えること**。

### 2.8 ドット化（出力工程）

| 項目 | 設計値 |
|---|---|
| 方式 | ベクター原画 → **80px**グリッド縮小 → アルファ2値化(128) → ポスタライズ(18) → pixelated拡大 |
| 粒度 | メイン80・アイコン96・将来のウィジェット48 |
| 禁則 | 半透明エッジのモヤ（=2値化で断つ）／グリッド非整数倍の常用表示 |

### 2.10 ポーズ体系（擬人化と動物性の使い分け・2026-06-11 発案者決定）

> 「実際は4足だけど擬人化したほうがなじみやすい。でも4足である場面は4足」

| ポーズ | 使う場面 | 造形 |
|---|---|---|
| **擬人化2足（基本形）** | ホーム・図鑑・詳細・巣立ち等「お友だちとして接する場面」 | 正面・腕2（脇から斜め24°・rx7.5×ry14.5・短く届かない）＋あんよ2（接地線から手前へ） |
| **4足歩行（quad）** | おさんぽ中など「動物としてふるまう場面」 | 横向き・進行方向は右・脚4本は奥2本を暗く/手前の前脚を一歩前に（歩行の互い違い）・目は1つ・しっぽは品種準拠 |

- 実装: `Art.petSVG(breed, stage, mood, 'quad')`。おさんぽ中オーバーレイは絵文字ではなく**飼っている子の4足歩行**を表示する
- 擬人化してよいのは**ポーズと手足の文法だけ**。服・直立二足歩行・5本指は禁止（動物であることを失わない）

### 2.9 時間設計（DEEP_DIVE §8）※静止画と同格の必須要素

- 呼吸: 3.4s・振幅2%（実装済み）／ squash先行のタッチ反応（実装済み bounce）
- まばたき4〜7秒1回・完全静止の「間」 → **[v1.1]**（キャンバス再描画が必要なため）

## 3. 検収チェックリスト（実装後に必ず通す）

1. **黒塗りテスト**: 6種を黒シルエット化 → 犬猫の別と可愛さが残るか（§1-1）
2. **3秒テスト**: 30種を名前なしで → 7割言えるか（§10）
3. 凸率・クラスタ凝縮度・瞳孔率・線階層比・色数の実測が§2の帯に入っているか
4. 44px縮小で目とほっぺが判別できるか（RUBRIC 指標1の小サイズ条件）
5. タンジェント事故・等間隔病・真円病がないか（§11目視）
