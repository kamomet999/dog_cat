# 犬種・猫種 マスターリスト（犬200・猫200＝計400・図鑑カタログ）

- 用途: 図鑑の全体カタログ。**無料60種（犬30・猫30）＋プレミアムで 犬200・猫200＝計400** まで拡張する元データ。
- 方針: 公認犬種/猫種（FCI/AKC/JKC/TICA 等）を中核に、人気の**色・毛柄・サイズ違い**も“集める楽しさ”として収録（猫は公認種が少ないので毛柄変種で200に）。
- ステータス: **ドラフト（あなたの「犬猫200ずつ」指示で再編）**。`★`=無料60種に既出（犬30・猫30）。数は概数、増減して確定。
- 次工程: 確定 → `breeds.js` データ化（art params/rarity/nature）→ `nanobanana-prompts.md` 再生成 → 画像生成。
