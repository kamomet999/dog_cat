# 犬種・猫種 マスターリスト（約600・図鑑カタログ）

- 用途: 図鑑の全体カタログ。**無料60種（犬30・猫30）＋プレミアムで約600種**まで拡張する元データ。
- 方針: 公認犬種（FCI/AKC/JKC 等）を中核に、人気の色・毛柄・サイズ違いの**バリエーション**も“集める楽しさ”として収録（猫は公認種が少ないため毛柄変種で数を確保）。
- ステータス: **ドラフト（あなたが「選んで」の指示で私が選定）**。`★`=無料60種に既出。数は概数で、追加/削除して確定する。
- 次工程: 確定 → `breeds.js` にデータ化（art params/rarity/nature 付与）→ `nanobanana-prompts.md` 再生成 → 画像生成。
