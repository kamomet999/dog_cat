## 第1部 人気の要素分解

人気作品に共通する造形要素を、出典付き・事実/推測を区別して構造的に列挙する。

### A. ベビースキーマ（Kindchenschema）— 学術的裏付け

**事実（複数の研究・解説で一致, スニペット）**: 「可愛い」と感じさせる赤ちゃん的特徴の束。具体的形態は
- 体に比して頭が大きい（人間の新生児で約4〜5頭身、成人で7〜8頭身）
- 顔面より脳頭蓋（額側）が大きい＝**広い額・高いおでこ**
- **目が大きくて丸い**
- **目鼻口が顔の中で低い位置にまとまる**（頭全体に対して下寄り）
- 小さい鼻・小さい口・小さい顎
- 丸い頬（ぷっくり）、短く太い四肢、丸い体
  - 出典: [かわいさ - Wikipedia](https://ja.wikipedia.org/wiki/%E3%81%8B%E3%82%8F%E3%81%84%E3%81%95) / [赤ちゃんの発達デザイン - babysigns.jp](https://www.babysigns.jp/article/14347.html) / [ベビースキーマ - pixiv百科](https://dic.pixiv.net/a/%E3%83%99%E3%83%93%E3%83%BC%E3%82%B9%E3%82%AD%E3%83%BC%E3%83%9E)

**事実（Glocker et al. 2009, スニペット）**: 17人の乳児顔を「額の高さ・顔の丸さ・目鼻口のサイズ」を操作して high/low/未操作の3版を作成。**high ベビースキーマ（丸い顔・高い額・大きい目・小さい鼻と口）が最も可愛いと評価され、養育動機を強く喚起**した。実際に測定した距離は Head Length, Face width, Forehead length, Face length, Eye width, Nose length/width, Mouth width, 鼻〜口間, 口〜顎間。
  - 出典: [Glocker 2009 / PubMed 22267884](https://pubmed.ncbi.nlm.nih.gov/22267884/) / [Frontiers 2014 baby schema](https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2014.00411/full)（本文は403で未取得）

**事実（眼サイズ研究, スニペット）**: 大きい目はあらゆる表情・年齢で、未操作・小さい目より可愛いと評価された。スニペットに現れた測定上の設定値として **eye size = 0.25W（顔幅の25%）, eye height（垂直位置）= 0.64H, eye distance = 0.43W** が挙がっていた。
  - 出典: [Eye Size Affects Cuteness - Frontiers 2021](https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2021.674456/full)（本文は403で未取得）
  - **推測/注意**: この 0.25W / 0.64H / 0.43W は刺激の一例値であり「最適値」と断定する原典記述は未確認。あくまで *人気帯の目安* として扱う。

### B. 日本のゆるかわ動物キャラ（ちいかわ/すみっコ/サンリオ系/カナヘイ等）

**事実（スニペット）**: 共通する造形語彙は
- 淡く優しい色（パステル）、丸くふわふわしたフォルム、シンプルで赤ちゃん的な愛らしさ
- 動物モチーフ＋「コロンと丸い」シルエット
  - 出典: [ちいかわとすみっコ 違い - mizuiro3216](https://mizuiro3216miz.com/sumikkogurashi-and-chikawa/) / [多摩美 すみっコ生みの親](https://www.tamabi.ac.jp/closeup/31068/)

**事実（スニペット）**: サンリオ系は「シンプルなデザインでありながらキャラごとに個性が出る工夫」。シナモロール＝クルクルしっぽ+青い瞳、ポムポムプリン＝ベレー帽（小物1点で識別性）。
  - 出典: [シナモロール vs ポムポムプリン - note/akih](https://note.com/akih/n/n3573e26b1294)

**推測（造形分析記事は表層的で、目の比率等の定量記述は得られず）**: 「同じ顔ベースに小物・色・1要素で差をつける」という方針は本アプリ（30種を同一顔ベースで描き分け）と整合。

### C. pixiv / X で人気の犬猫イラスト & デフォルメの原則

**事実（イラスト講座, スニペット）**: 「ゆるい/可愛い」方向のデフォルメ原則は
- **目も口も小さめ……ではなく、目を大きく・口を小さく**（クール系は目を細く、可愛い系は目を大きく丸く）
- 猫=顔は横長・耳は正三角形・鼻と口は小さめ・ヒゲで猫らしさ／犬=顔と耳は縦長・鼻と口は大きめ
- もふもふを増やすほど可愛く、シャープにするほど凛々しい
  - 出典: [上手な犬・猫の描き方 - CLIP STUDIO](https://www.clipstudio.net/oekaki/archives/151584) / [リアルな猫とゆるい猫の描き分け - CLIP STUDIO TIPS](https://tips.clip-studio.com/ja-jp/articles/2365) / [動物キャラをかわいくデフォルメ - いちあっぷ](https://ichi-up.net/2018/068)

**事実（ミニキャラ/2頭身講座, スニペット, 複数で一致＝最も定量的）**:
- **目は顔の下から約1/4の位置に置くと可愛い**（高すぎ低すぎはバランス崩れ）
- **目と目の間隔は目1つ分**空ける
- **顔のパーツは顔全体の下半分のスペースにまとめる**（やや下に寄せる）
- ミニキャラは頭が大きい分**目も比例して大きく**、黒目を大きく・ハイライトを入れシンプル化
  - 出典: [ミニキャラの描き方 - パルミー](https://www.palmie.jp/lessons/94) / [2頭身を可愛く描く3ステップ - kiri-hana](https://kiri-hana.com/minikyara-kakikata/) / [ミニキャラの目 - kiri-hana](https://kiri-hana.com/minikyara-me/) / [ミニキャラの描き方 - CLIP STUDIO](https://www.clipstudio.net/oekaki/archives/153412)

## 要約レポート

### 指標一覧（測り方つき）
1. **目径/顔幅比** = 目の直径÷顔幅。目安22〜30%。
2. **目の垂直位置（下寄り度）** = 目中心の顔下端からの距離÷顔高。目安25〜45%（小さいほど下＝幼い）。
3. **両目間ギャップ/目幅** = 目エッジ間隔÷目直径＝「目何個分」。目安0.8〜1.2個分。
4. **鼻+口の小ささ** = 鼻先〜口下端の縦幅÷顔高。目安≦18%。
5. **頬の存在と位置** = ほっぺ直径÷顔幅≥8% かつ ほっぺ中心が目より下。
6. **輪郭視認性** = 閉じた輪郭線の線幅÷顔幅≥2.5%（最小44pxで残ること）。

### 現行画風の最大の弱点トップ3＋数値修正案
1. **【最大の弱点】目・顔パーツが顔の中で高すぎる（中央寄り）**。指標2が犬43.8%/猫42.4%で、ベビースキーマ規範「下から25〜33%」より上＝幼児性を取りこぼし。
   → **修正**: 顔パーツ群を一括 **+6〜8px 下げる**（頭円cy92は固定）。犬: 目cy98→106・鼻口111→117・口元115→120・ほっぺ116→122。猫: 目cy99→106・鼻口112→118・口元113→119・ほっぺ116→121。これで犬35%/猫35%に入り、同時に「広い額」も獲得。
2. **目の大きさが下限寄りで伸びしろ未活用**（25〜26%は合格だが中央25%＝刺激例値の下限付近）。
   → **修正（任意）**: `eyeScale 1.05→1.12` で直径26〜28%へ。必ず修正1（下げ）と同時に。
3. **両目間隔が規範ど真ん中よりわずかに狭い**（0.90個分）。実害は小さい。
   → **修正（任意・低優先）**: 目中心を `cx77→75 / cx123→125` で1.0個分に。

### 信頼性の注記
- 実務系（イラスト/LINE 講座）は数値が具体的で複数ソース一致＝**信頼度高**。
- 学術系（Glocker 2009、眼サイズ研究、入戸野論文）は **本文PDFがWebFetch 403で全滅**したため **WebSearchスニペット依拠**。特に「0.25W/0.64H/0.43W」は刺激の一例値であり「最適値」かは**原典未確認＝不明**。方向性（大きい目・低い位置・丸顔・小さい鼻口がより可愛い）は複数ソースで一貫。
- 本アプリの「もちもち一体ブロブ」のため頭身比は独立測定不能＝指標2・5で代替（設計判断）。
