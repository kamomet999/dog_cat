# ゲームデザイン仕様 — 生存システム（いのちと家出）

- 作成日: 2026-06-12 ／ 最終改訂: 2026-06-12（v5: えさ自動給餌×さんぽ課題） ／ 対象: v1.0（6/22凍結前）
- 発案者決定の積み上げ:
  1. 「ごはんがないと死ぬ・スマホを触っていない時間がないと死ぬ」
  2. 「なにがごはんになるかが大事」
  3. 「ごはん怠り→死 ／ さんぽ怠り→逃げる（家出）」
  4. 「ごはん＝スマホを触らない時間で餌が貰える・ストック制・自動給餌。さんぽ＝学習や運動など"いい時間"・リミット長め」

---

## 1. コンセプト

> **この子は「あなたがスマホを置いた時間」を食べて生き、「あなたのいい時間」に寄り添って暮らす。**

| | **ごはん（えさ）** | **さんぽ（いい時間）** |
|---|---|---|
| 源泉 | スマホを触らない時間 | 学習・読書・運動など自分で決めた課題の時間 |
| 仕組み | えさとして貯まる→**ストック→自動給餌**（何日ぶんか先まで勝手に食べる） | 課題セッションでゲージ回復（**失敗なし**） |
| 怠ると | **おほしさま（死）** — リミット短め・ストックが日数バッファ | **家出（旅に出る）** — リミット長め |
| 通知像 | YouTube/SNSを見ている画面の上に「えさがなくなりそう…」が降りてくる → **スマホを閉じる動機** | 「いっしょのじかんがほしいな」 |

## 2. えさ（ごはん）

```
獲得（＝スマホを置いた時間）
  ごはんさがし（ふせセッション・主獲得）: 30分=2 / 1h=4 / 2h=8 えさ
  受動回復（アプリを閉じている時間）   : 0.1えさ/h（2.4/日・報酬と同じ24h上限）
  コイン購入（コイン=放置時間の蓄積）   : 20コイン/1えさ
  ストック上限21（約1週間ぶん）・初期6

消費（自動給餌）
  おなか<40 になったら自動で1えさ（おなか+60）。おなか減衰 -100/12h ≒ 1日3えさ
  「てであげる」と なかよし+4・きげん+8 のボーナス（自動にはない・絆の演出）

死（リミット短め・予告つき）
  ストック切れ→おなか0→ いのち -100/24h ／ おなか>30で +100/48h 回復
  完全放置: ストック6＋微獲得で約3日は自動でしのぐ → その後 おなか0から24hで おほしさま
  （= アプリを忘れて数日でおわかれ。いのちハート5・しょんぼり表現・通知で必ず予告）
```

## 3. さんぽ（いい時間・課題セッション）

```
種類: どくしょ / えいご / うんどう / ジョグ / しゅうちゅう（自分の課題）
長さ: 15 / 30 / 60 分 → ゲージ +35 / +50 / +80（上限100）
性質: 失敗なし。開始したら他アプリ（Kindle・英語アプリ等）を使ってOK。
      途中でやめても「またこんど」（ペナルティなし）
ゲージ: -100/168h（1週間で空）。0が72時間つづくと家出 ＝ 計10日サボると旅に出る
家出: 「あたらしい家族をさがしに行く」（死ではない・責めないトーン・図鑑非登録・runaways記録）
```

- **v1は自己申告（オナーシステム）**。「特定アプリを開いている時間の自動計測」は
  Android=UsageStats特別権限／iOS=ScreenTime API（審査ハードル高）のため **v2研究テーマ**として明記。
  ジョグ連携（歩数/GPS）も同枠

## 4. 通知（ドゥームスクロールへの割り込み）

> 発案者イメージ: 「YouTubeやSNSを見ていたら警告が出てくる。おなかすいてるからスマホ閉じてごはんとさんぽさせなきゃ、と動機づけたい」

- 仕組み: アプリがバックグラウンドへ移行する瞬間に `dangerForecast` で**枯渇時刻を予測**し、ローカル通知を予約（サーバー不要）。
  他アプリの使用中でも、**その画面の上に**バナーが降りる＝ドゥームスクロール中の割り込みが成立
- 文言（ペット主語・責めない）:
  - えさ予告: 「えさが なくなりそう…🍚 スマホを おいて ごはんさがしに いこう」
  - さんぽ予告: 「いっしょの じかんが ほしいな🐾 とおくへ いっちゃう まえに」
  - いのち警告: 「ぐあいが わるいみたい…💧」
- Androidは重要度最高のチャンネル（凍結前TODO）／iOSは標準バナー
- 予測は**ストックを考慮**（えさがあるかぎり空腹予告は出ない＝通知が来た時は本当に必要なとき）

## 5. おわかれ／旅立ちの設計

- どちらも**責めないトーン**・赤/警告アイコン禁止・図鑑には登録されない（図鑑=最後まで育てた記録）
- おほしさま🌟「いっしょにすごしたじかんはきえないよ」／旅立ち🎒「どこかでげんきにしているよ」
- 記録: せってい画面に件数。**課金での復活・延命は永久になし**（DESIGN.md 変えない約束）
- 直後に無料で新しい子をおむかえ（立ち直りを妨げない）
