# マーケティング分析 — いぬねこ図鑑

- 作成日: 2026-06-10
- 調査方法: AIエージェントによるWeb調査（出典付き）＋戦略分析
- 結論を先に: **方向性の大転換は不要。ただし「何で勝つか」を3点に絞って磨く**

---

## 1. 結論（エグゼクティブサマリ)

1. 「スマホ断ち×かわいいキャラ育成」は**2025年に世界的に実証済みの激戦区**（Focus Friendが米App Store総合1位）。市場があることは証明されているが、メカニクスだけでは勝てない
2. 本企画の固有部分=「**犬・猫の実在品種図鑑（30種）を埋める**」を中核にした日本語デトックスアプリは、調査の範囲では**存在しない（空白地帯）**
3. 最大の直接競合は日本製「**スマホをやめれば魚が育つ**」（スマやめ）。レビュー1.5万件超で定着済みだが、**広告過多・コンプに課金必須**が最大の不満点 → 本企画の「広告ゼロ・無課金で全種コンプ可能」はそのまま刺さる差別化になる
4. リスクは参入障壁の低さ（大手がテーマのスキン替えを高速で行う）。**速く出して、日本語の「ゆるかわ」と図鑑の情緒で先にファンを作る**ことが防御

### 市場の追い風（日本・2024〜2026）
- デジタルデトックスアプリ市場は約5.3億〜9.8億ドル（2025年・調査により幅）、CAGR 20%超の予測
- 豊明市「スマホ1日2時間」条例（2025年10月）、「スマホ認知症」外来開設など社会的関心が高い
- MMD調査で約8割が「スマホ依存」を自覚。SHIBUYA109 lab.のZ世代トレンド2026に「**アテンション・デトックス**」

## 3. ポジショニング — 「何で勝つか」を3点に絞る

```
            かわいさ・情緒
                 ▲
   Focus Friend  │  ★いぬねこ図鑑
   （キャラ1体） │  （30種の図鑑を埋める）
                 │
 ◀──────────────┼──────────────▶
 ツール寄り      │       コレクション寄り
                 │
   Forest        │  スマやめ
   （木・実用）  │  （魚・広告多）
                 ▼
            実用・ストイック
```

### 勝ち筋3点（これ以外は足さない）

1. **「全30種、無課金・広告ゼロでコンプできる」を公約にする**
   - スマやめの最大不満（広告・コンプ課金）の正反対。ストア説明文・スクショに明記する
   - 収益化する日が来ても「図鑑コンプは無料」の線は守る（着せ替え・応援課金側で取る）
2. **図鑑の情緒で差別化する**
   - Forestの弱点は「木に愛着が湧きにくい」。犬猫の品種＋「はじめて出会った日」＋殿堂入りで、**ポケモン図鑑的な達成感**と**ペットへの愛着**を両立させる（実装済み）
   - 品種は30→50→100種と増やせる構造（プロシージャルSVG）が長期の伸びしろ
3. **「がんばらないデトックス」のトーン**
   - 競合は「集中・生産性」文脈（勉強・仕事）が主戦場。本企画は「**スマホとちょっと距離を置く**」の生活文脈で、失敗しても死なない・責めない設計。Z世代「アテンション・デトックス」の言葉と相性が良い

## 4. 方向性のアドバイス（変える/変えない）

### 変えなくて良いもの
- **コアメカニクス**（おさんぽセッション＋放置成長）: Forest型は実証済みの型。新規性をメカニクスで出す必要はない
- **完全ローカル・無広告**: 最強の差別化かつ審査も最速。維持
- **犬猫×図鑑**: 空白地帯。ねこあつめ・Pokémon Sleepが日本での受容を実証済み

### 変える・足すべきもの（優先順）
1. **【v1.1最優先】ホーム/ロック画面ウィジェット**: 「アプリを開かずにペットの様子が見える」はデトックスと完全に整合し、Widgetable（月100万DL）が需要を実証。毎日視界に入る=最強のリテンション
2. **【v1.0でも軽く】シェア画像の自動生成**: おさんぽ成功・図鑑コンプ時に「○時間スマホをはなれて、コーギーが育ちました」カードを保存/共有できるように。広告費ゼロの個人開発では**シェアが唯一の獲得チャネル**
3. **【ASO】タイトル・キーワード**: 「いぬねこ図鑑」単体では検索に乗らない。`スマホ依存 / デジタルデトックス / 集中 / 犬 / 猫 / 育成 / 放置` を副題・説明文に織り込む（listing.mdは対応済み。リリース後に検索順位を見て調整）
4. **【v2検討】保護犬猫への寄付連携**: Pomokitten（収益10%寄付）の型。「犬猫アプリ」としてのPR文脈（メディア・SNS）が作りやすい。収益化を始めるときの「感じの良い」理由付けになる

### やらないと決めるもの
- Widgetable型の「画面を見せる」方向への転換（市場は大きいがコンセプトが崩壊する）
- Finch型セルフケア・サブスク（コンテンツ量で勝負になり個人開発では不利）
- グループ連帯責任（Flora型）・ランキング: ソーシャルはv2以降。まず1人で完結する気持ちよさ

### 重要なリスクと現実的な見立て
- **テーマの参入障壁は低い**。当たれば大手が「犬猫スキン」を出してくる。先行者の防御は「図鑑の品種数と情緒の積み上げ」と「日本語コミュニティ」しかない → だからこそ**前倒しリリースは正しい**
- 個人開発アプリの最大の死因は品質ではなく**発見されないこと**。リリース＝ゴールではなく、シェア画像・ASO・X/TikTokでの「育った報告」が回り始めるまでが勝負
