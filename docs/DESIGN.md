# デザイン方針書 — いぬねこ図鑑

- 作成日: 2026-06-10
- 根拠: 競合ビジュアル調査（出典は調査レポート参照）＋現行実装の分析
- 結論: **「暖色×ゆるかわ×引き算」路線を確定。** 緑系（自然・規律）には行かない

---

## 1. ポジショニングから導くデザインの大方針

調査で確認できた市場の構図：

| 陣営 | 代表 | 訴求 | 状況 |
|---|---|---|---|
| 緑・自然・ミニマル | Forest / Flora / Focus Plant | 規律・集中・自然メタファー | 老舗が密集するレッドオーシャン |
| **暖色・キャラ・世話** | **Focus Friend / Finch** | 愛着・「この子のために」 | **近年の勝者**（全米1位・ARR$30M） |

Focus Friendの成功要因はレビュー言説で「**罪悪感ではなく、かわいい存在への責任感**で行動が変わる」こと。Finchは「かわいさは装飾ではなく感情の設計（emotional engineering）」。本アプリの現行デザイン（クリーム×オレンジ・丸ゴシック・ひらがな）は**すでに勝ち筋側にいる**ため、転換ではなく徹底を行う。

### デザイン原則（3つ）

1. **この子のために置く** — UIは常にペットの感情を主語にする（「失敗しました」ではなく「しょんぼりしてる…」）
2. **ゆるくて、責めない** — 高彩度・赤・警告アイコン・数字の詰め込みを避ける。低彩度×余白×丸み
3. **生きてる感** — 静止画を置かない。呼吸・まばたき・反応（Finchの定石）

## 2. カラーシステム

### ひる（メイン）パレット — 現行を維持・確定

| トークン | 値 | 用途 |
|---|---|---|
| `--bg1/--bg2` | #fdf6ec / #f6e7d8 | 背景（クリーム→ベージュ） |
| `--accent/--accent-d` | #ff9a76 / #f47b54 | 主ボタン・強調（このアプリで唯一の高目立ち色） |
| `--ink/--ink-soft` | #4a3f35 / #8a7d6d | 文字（真っ黒を使わない） |
| `--good/--warn/--bad` | #7ac77a / #e8a13a / #e06464 | ステータスバーのみ。**UIの警告には使わない** |
| `--coin` | #f2b134 | 報酬 |

### よる（おさんぽ）パレット — 既存の深緑を正式トークン化

おさんぽ中画面の `#2e4a3a → #1d3328` は「夜の散歩」の世界観として正式採用。2025-26トレンド（ウェルネス系はダークが第一級）に対する**v1.0の回答**はこの画面に限定し、アプリ全体のダークモードはv1.1で対応する（Pokémon Sleepですらダーク非対応が批判された、という調査結果に基づく優先度付け）。

### ルール
- 高彩度色は同一画面に1つまで（アクセントのオレンジ）
- 影は黒ではなく茶系（`rgba(120,90,60,…)`）— 既存実装を維持
- 新しい色を追加する前に既存トークンで代替できないか確認する

## 3. キャラクター規定（ゆるかわ文法）

調査で複数ソース確認できた「ゆるかわ」構成要素と、本アプリでの実装状態：

| 要素 | 根拠 | 実装状態 |
|---|---|---|
| 頭身約2:1・丸いフォルム | ベビースキーマ（保護本能の自動刺激） | ✅ 現行: 頭r48 vs 体r42〜50 |
| つぶらな目＋ハイライト | 同上 | ✅ 現行 |
| **ほっぺの赤み（常時）** | 同上 | ✅ **今回追加**（通常時うっすら・喜ぶと濃く・しょんぼり時は消える） |
| **アイドル時の「生きてる感」** | Finchの emotional engineering | ✅ **今回追加**（3.4秒周期のゆっくり呼吸） |
| なでると反応 | 同上 | ✅ 現行（bounce/wiggle） |
| 手描きの線の揺らぎ | ゆるかわ文法（ラフ線が緊張を解く） | ⏩ v1.1検討。プロシージャルSVGに輪郭ゆらぎを足す実験 |
| 要素の引き算 | ねこあつめ「複雑さを入れない」 | ✅ 維持。装飾追加の誘惑に抵抗する |

## 4. UIコピー（ボイス&トーン）

- ひらがな優先・漢字は最小限（現行どおり）
- ペットの感情を主語に: ❌「セッション失敗」→ ⭕「ちょっとしょんぼりしてる…」
- 命令しない: ❌「スマホを置いてください」→ ⭕「スマホをふせて、まっててね」
- 失敗時も次を約束: 「つぎはきっとだいじょうぶ！」（実装済み）

## 5. アプリアイコン

- **採用: 柴犬の顔アップ（笑顔・ほっぺ）**。調査結果「カジュアル・コンシューマー系では『フレンドリーなキャラ顔』の勝率が高い」「最小サイズでの判読性・支配的要素1つ」に合致
- 肉球マークは**ブランドモチーフ**としてスプラッシュ・空状態・装飾に継続使用（`assets/icon-paw-variant.png` に保存）
- 公開後、ストアのA/Bテスト（Play実験/PPO）で 顔 vs 肉球 を検証する（アイコン変更は2桁%のCVR差が出る変数）

## 6. ストアスクリーンショットの公式（調査ベース）

- **最初の3枚が約70%のコンバージョン重量・閲覧7秒** → 1枚目に機能説明を置かない
- 構成は **Value → Usage → Trust**:
  1. 「スマホを置くと、育つ。」（ホーム画面＋大コピー）
  2. 「30分の おさんぽから」（おさんぽ画面）
  3. 「ぜんぶ無料で 30種コンプ」（図鑑）
  4. 使い方3コマ / 5. 「広告ゼロ・登録不要」
- コピーは**1行・3〜5語・サムネイルで読める太さ**（拡大して見る人は4%しかいない）
- iOS=世界観ポスター寄り / Google Play=明るい背景＋実UI寄りに作り分け

## 7. 今回実装した変更（クイックウィン）

1. ほっぺの赤みを常時表示に（mood連動: 通常0.35-0.38 / 喜び0.55-0.6 / 悲しみ非表示）
2. ペットのアイドル呼吸アニメーション（CSS・3.4s周期・transform-origin足元）
3. アプリアイコンを肉球→柴犬顔に変更（全サイズ再生成済み）
