## 4. 「おまかせ承認」で動く構成

- 技術: TypeScript + React + Vite ＋ PWA（Zustand / IndexedDB / Service Worker通知 / Vercel配信）
- MVP: 犬猫1匹を選び育成。4ステータス時間減衰＋世話4種＋**オフライン進行＋放置報酬**＋成長3段階＋通知1種。
- OSSライセンス: React/Vite/Zustand/idb-keyval/Framer Motion いずれも MIT（商用可）。**アート素材のみ権利確認が必要**。

## 5. コスト・期日・必要アカウント

- 直接費: **推奨案は実質ゼロ**（内製・無料ホスティング枠）。生成AI素材を有料プランで作る場合のみ少額。
- 期日: MVP公開まで **4〜6週**（マイルストーン M1触れるプロト→M2オフライン進行→M3通知→M4公開）。
- 必要アカウント: Vercel/Netlify（無料）。通知の実機検証に iOS/Android 実機。**課金・外部公開・2FAを伴う登録は社長承認後に実施**。

3. **スコープ膨張で未完** → MVP範囲（§2.2）を死守。多頭飼い・課金・SNSは v0.2 以降。
